# 🩸 DonorX Blood Donation Analytics Dashboard

## 📌 Project Overview

DonorX is an interactive Power BI dashboard developed to analyze blood donation data and provide meaningful insights into donor behavior, blood inventory, campaign performance, and request fulfillment.

This project is an extension of my previous **DonorX Blood Donation Application**, where I transformed operational data into an analytics solution for better decision-making.
